How to sign in

Each app sign-in uses a magic link. Enter any of these demo emails and the link arrives in the inbox you enter:

• Club admin · admin.demo@unityfcacademy.co.uk

• Coach (U10 Reds) · coach.morris@unityfcacademy.co.uk

• Parent (of Morgan) · parent.morganjones@example.com

The email addresses aren’t real inboxes yet — in Supabase Studio → Authentication → Users you can generate a magic link for any of them and paste it into the browser to sign in.

Safeguarding — proven at the DB

Coach sees 14-day wellbeing patterns (Jude’s flag appears). Parent sees Morgan’s digest and toolkit. Neither sees raw mood check-ins — only Morgan does. That boundary is enforced by RLS policies on Supabase, not app logic; see tpm/supabase/rls_safeguarding_tests.sql in the repo for the nine-test suite.
